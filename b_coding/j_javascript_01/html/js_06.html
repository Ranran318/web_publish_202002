반복문 이해하기

  • while: 하나의 조건을 시작해서 그것을 반복처리하게 만드는 기능
    최초시작, 판단조건, 수행증감처리하는 부분이 다른 곳에 배치
  • do-while: 한번은 무조건 수행하고 이후에 while과 동일하게, 반복처리 기능
  • for:하나의 조건을 시작해서 그것을 반복처리하게 만드는 기능 최초의 시작, 판단조건, 수행증감여부의 구문이 한군데 모여있다.
  • for-in: for문과 기본 기능은 같지만, 객체형태에서만 사용할 수 있는 기능 (배열x)
  • forEach: for문과 유사한 기능이지만, 해당기능을 각각 수행하는 처리능력을 가지고 있다.
